Week 11 - Progress Update

What We Achieved This Week

This week, as part of our Satellite Orbit and Event Predictor (SOEP) project, we've been focused on implementing some exciting new improvements and fixed some bugs. Here's what we've accomplished:

  • Moved backend functionality to frontend After implementing the backend, we soon realized that it takes longer to retrieve the satellite data from the database then just generating it on the go. Therefore we decided to move the calculations from the backend directly to the frontend. So, now calculations are going to be made on client-side.
  • Threading in the frontend To make the asynchronus calculations faster we implemented threading in the frontend
  • Fixed bugs We fixed several bugs in the frontend
  • Added new features We added some new UI features, for example added timezones to the settings, made the satellites smaller and started designing the mobile popups
