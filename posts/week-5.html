Week 5 - Progress Update

What We Achieved This Week

This week we focused on setting up the server, enhancing security measures, and hosting our Satellite Orbit and Event Predictor (SOEP) project website. Here’s a summary of what we accomplished:

  • Server Setup: We fully configured our server, setting up a robust framework for our project’s needs.
  • Domain Setup: We successfully registered and set up the domain that will host our application.
  • DDNS Integration: Added dynamic DNS to ensure the website remains accessible even if the public IP changes, enhancing reliability.
  • HTTPS Upgrade: Implemented HTTPS using Nginx, increasing the security of data transmissions to and from our website.
  • Hosting on Local Server: Our website/app is now hosted on a local server, making it accessible securely via https://soep.tech/.
